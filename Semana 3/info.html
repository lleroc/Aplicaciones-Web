Introducción

En la programación web con C# y ASP.NET, el envío de datos entre el cliente y el servidor se realiza mediante formularios HTML y se pueden gestionar a través de los métodos HTTP GET y POST.

Envío de Variables por Formulario

Los formularios en una aplicación web permiten a los usuarios ingresar datos y enviarlos al servidor. 

<form method="post" action="/Home/RecibirDatos">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required>
    <br>
    <label for="edad">Edad:</label>
    <input type="number" id="edad" name="edad" required>
    <br>
    <button type="submit">Enviar</button>
</form>

Método GET

El método GET se usa para recuperar datos del servidor enviando información en la URL. 

[HttpGet]
public IActionResult RecibirDatos(string nombre, int edad)
{
    ViewData["Mensaje"] = $"Nombre: {nombre}, Edad: {edad}";
    return View();
}

Ejemplo de URL:

http://localhost:5000/Home/RecibirDatos?nombre=Juan&edad=25

Método POST

El método POST se usa para enviar datos al servidor de forma segura en el cuerpo de la solicitud HTTP. 

[HttpPost]
public IActionResult RecibirDatos(string nombre, int edad)
{
    ViewData["Mensaje"] = $"Nombre: {nombre}, Edad: {edad}";
    return View();
}

Diferencias entre GET y POST

Característica GET POST
Seguridad Menos seguro (visible en URL) Más seguro (oculto en el cuerpo de la solicitud)
Capacidad de datos Limitada por la URL No tiene limitaciones
Uso principal Consultas y recuperación de datos Envío y modificación de datos
